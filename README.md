# Bangumi 条目分享卡片

[![GitHub Raw](https://img.shields.io/badge/GitHub%20Raw-%E5%AE%89%E8%A3%85%E8%84%9A%E6%9C%AC-2ea44f?logo=github)](https://raw.githubusercontent.com/aronnaxlin/bgm-cards/main/userscript/share-card.tampermonkey.user.js)
[![超合金组件](https://img.shields.io/badge/%E8%B6%85%E5%90%88%E9%87%91%E7%BB%84%E4%BB%B6-6417-ff69b4?logo=bangumi)](https://bgm.tv/dev/app/6417)

为 Bangumi 条目页（`/subject/{id}`）一键生成精致的 **PNG 分享卡片**，可下载或复制到剪贴板。

![pic](./assets/pic.png)

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)（或兼容的脚本管理器）。
2. 点击上方 **GitHub Raw** badge 安装脚本。
3. 进入任意 Bangumi 条目页即可使用分享卡片功能。
